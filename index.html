SIMPLIFYING CONCEPTS

The Internet is a structure made up of millios of computers whose users can communicate with each other and share information.

The World Wide Web (WWW) gives Internet users easy access to information on the Internet.

HTML is Hypertext Markup Language and it is the main, or most common, type of document on the web. There are other types such as text, images, videos, and music. HTML was designed to describe the contents of a web page in a very general way.

HTTP is Hypertext Transfer Protocol that sends requests to a srver.

Computers are stupid because they can only do what programmers tell them to do. The programmers have to write in a language that the computer can read. This is referred to as CODING.

Let's take a look at some of the basic HTML vocabulary.

  • ELEMENT: Everything within a set of opening and closing tags.
  • TAG: Turns on a feature which is indicated by an opening tag, and turns off a feature indicated by a closing tag.
  • INLINE ELEMENTS: Tags that do not require a closing tag.
  • BLOCK ELEMENTS: Tags that require a closing tag. Block Elements are similar to an invisible box being placed around the text.
  • ATTRIBUTE: A property of an HTML element.
  • There is a tag to do BOLD text.
  • There is a tag to do emphasis such as italics.
  • There are also tags to do line breaks and paragraphs.

    There are many others. There are tags to do almost everything. After all, computers are stupid and will only do what we tell them to do, so we have to talk to them in a language they can understand--HTML
